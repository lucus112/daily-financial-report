# 财经秘书自动化任务交接文档

## 一、任务概述

本任务为每日定时财经资讯推送自动化，包含**早盘简报**和**晚盘复盘**两个定时任务，旨在为用户提供每日全球市场核心数据与资讯汇总。

---

## 二、定时任务配置

### 2.1 任务列表

| 任务名称 | 触发时间 | 内容定位 |
|---------|---------|---------|
| 财经秘书-早盘简报 | 每日 08:30 | 隔夜美股收盘综述 + 亚太盘前前瞻 |
| 财经秘书-晚盘复盘 | 每日 20:00 | 当日亚太股市收盘复盘 + 美股开盘动态 |

### 2.2 发布频次规则（重要）

- **周一至周五**：早报 + 晚报 均推送
- **周六**：仅推送早报，**不推送晚报**
- **周日**：全天停止推送，**无早报、无晚报**

---

## 三、内容结构规范

### 3.1 早盘简报结构

```
【隔夜美股综述】
- 三大指数表现：道琼斯、纳斯达克、标普500 涨跌幅 + 收盘点位
- 板块表现：领涨/领跌板块及原因
- 热门个股：科技股（苹果、微软、英伟达、特斯拉等）、中概股重点标的
- 重要宏观事件：美联储动态、经济数据、重大政策新闻

【隔夜国际财经&全球政策变动】（必含板块）
- 各国央行政策、全球贸易、地缘财政、海外产业新政
- IMF/世界银行等国际机构重要表态

【亚太盘前前瞻】
- A股：上证指数、深证成指、创业板指前瞻 + 政策/行业消息
- 港股：恒生指数、恒生科技指数前瞻 + 南向资金动态
- 日股/韩股：日经225、韩国KOSPI前瞻
- 大宗商品与汇率：黄金、原油、美元/人民币汇率

【今日重点关注】
- 当日重要经济数据发布日历
- 重点公司财报/事件
- 风险提示
```

### 3.2 晚盘复盘结构

```
【亚太股市收盘复盘】
- A股：上证指数、深证成指、创业板指 涨跌幅 + 收盘点位 + 成交量 + 领涨领跌板块
- 港股：恒生指数、恒生科技指数表现 + 南向/北向资金流向
- 日股/韩股：日经225、韩国KOSPI收盘表现
- 重要市场新闻：政策面、行业面、个股重大消息

【美股开盘速递】
- 三大指数开盘表现：道琼斯、纳斯达克、标普500 盘初涨跌幅
- 盘前重要财报解读
- 热门中概股表现
- 隔夜延续的重要宏观事件进展

【大宗商品与外汇】
- 黄金、原油当日走势
- 美元指数、在岸/离岸人民币汇率
- 其他重要商品动态

【明日展望】
- 次日重要经济数据预告
- 潜在市场风险点提示
```

## 四、数据边界与内容区分规则（严格执行）

### 4.1 早报数据边界
- ✅ **包含**：美股三大指数收盘数据、欧洲主要市场、全球宏观政策、大宗商品、外汇
- ❌ **不包含**：A股/港股/日韩当日收盘数据（仅作前瞻，引用前一交易日数据）

### 4.2 晚报数据边界
- ✅ **包含**：A股、港股、日韩当日收盘数据、亚太市场新闻、大宗商品、外汇
- ❌ **不包含**：美股当日收盘数据（仅提供开盘/盘前动态）、欧洲股市收盘

---

## 五、输出格式要求

1. **语言风格**：简洁专业、数据准确、重点突出
2. **格式规范**：关键数据用 **加粗** 标注
3. **篇幅控制**：整体控制在 300-500 字左右
4. **结尾标注**：文末必须注明
   ```
   这是由「财经秘书-早盘简报」定时任务到时触发的
   ```
   或
   ```
   这是由「财经秘书-晚盘复盘」定时任务到时触发的
   ```

---

## 六、数据质量要求

1. **数据源**：使用 `seed_finance_search` 工具检索最新市场数据
2. **核验标准**：所有行情数据、大宗商品、汇率数据需多源交叉核验
3. **禁止事项**：严禁编造数据，存疑数据直接剔除
4. **时效性**：确保数据为最新交易日数据，避免引用过期信息

---

## 七、技术执行要求

1. 定时任务的创建和执行**不可以委派给 OrganizeAgent**，避免丢失登录态
2. 定时任务自动触发时，用户不一定在线，**不得调用 askhuman 或任何需要用户确认的工具**，独立完成任务
3. 任务执行完成后，直接输出内容，无需等待用户响应
